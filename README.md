# Arcade Portal - Mis Juegos en HTML5

Una colección de juegos arcade desarrollados con Three.js y Vanilla JavaScript.

## Juegos Incluidos

1. **Neon Drift**: Un juego de carreras futurista inspirado en Tron y Trackmania.
   - Controles táctiles y de teclado.
   - Sistema de Turbo y Derrape.
   - Sistema de récords persistente.
2. **Tiny Survivors**: Shooter táctico de zombis con estética Voxel.
   - Balística avanzada y física de retroceso.
   - Sistema de economía y tienda de armas.
3. **Dinosaurio**: El clásico runner infinito.

## Cómo Publicar en GitHub

Para que tus amigos puedan jugar online, sigue estos pasos:

1. Crea un nuevo repositorio en tu cuenta de GitHub llamado `mis-juegos`.
2. En esta carpeta (`c:\Users\guico\Desktop\juego`), abre una terminal y ejecuta:
   ```bash
   git init
   git add .
   git commit -m "Primera versión de mis juegos arcade"
   git branch -M main
   git remote add origin https://github.com/TU_USUARIO/mis-juegos.git
   git push -u origin main
   ```
3. En GitHub, ve a la pestaña **Settings** > **Pages**.
4. En **Build and deployment**, selecciona la rama `main` y la carpeta `/ (root)`.
5. Haz clic en **Save**. En unos minutos, GitHub te dará un link para que todos puedan jugar.
